Materi Latihan Java: Array - Pemula

Petunjuk:
- Jawaban ditulis langsung dalam bentuk kode Java.
- Simpan semua jawaban dalam satu file Java, misalnya: LatihanArray.java
- Jalankan kode untuk memastikan hasilnya sesuai.

========================================
SOAL 1:
Buat sebuah array integer bernama "angka" dengan 5 elemen, isi dengan angka: 10, 20, 30, 40, 50. Tampilkan semua elemen menggunakan loop for biasa.

========================================
SOAL 2:
Buat array string bernama "buah" yang berisi: "apel", "jeruk", "mangga". Tampilkan elemen kedua dari array tersebut.

========================================
SOAL 3:
Buat array integer bernama "nilai" dengan 4 elemen. Isi nilai ke-3 dengan angka 75, kemudian tampilkan elemen ke-3 tersebut.

========================================
SOAL 4:
Buat array double bernama "harga" berisi 3 elemen: 1000.5, 2500.75, 3000.0. Tampilkan semua elemen menggunakan loop for-each.

========================================
SOAL 5:
Buat array 2D integer 3x3 bernama "matrix" dengan angka 1 sampai 9. Tampilkan seluruh matrix menggunakan nested loop.

========================================
SOAL 6:
Buat array char bernama "huruf" berisi 'A', 'B', 'C', 'D'. Ubah elemen ke-2 menjadi 'Z' dan tampilkan seluruh array.

========================================
SOAL 7:
Buat array boolean bernama "status" dengan 3 elemen: true, false, true. Tampilkan panjang array menggunakan properti length.

========================================
SOAL 8:
Buat array string "hari" berisi nama-nama hari dalam seminggu. Tampilkan elemen terakhir dari array tersebut.

========================================
SOAL 9:
Buat array integer "numbers" dengan 5 elemen, isi sesuai pilihanmu. Hitung dan tampilkan jumlah semua elemen menggunakan loop.

========================================
SOAL 10:
Buat array 2D integer "tabel" 2x3 dengan angka bebas. Ubah elemen baris pertama kolom kedua menjadi 99, lalu tampilkan seluruh tabel.
